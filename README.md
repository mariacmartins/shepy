### Descrição: 
Repositório contendo o conteúdo ministrado nas aulas do projeto **shePY**.

### O que é o shePY:
O **shePY** (she + Python)é um projeto voluntário de ensino de programação focado em mulheres, visando o empoderamento feminino na área.
